# Filmų katalogas
## Projekto tikslas
Sukurti C++ programą, skirtą filmų katalogo peržiūrai bei modifikavimui.
## Pasirinktos temos aprašymas
Filmų katalogas leidžia saugoti informaciją apie filmus: pavadinimą, režisierių, išleidimo metus, žanrą, įvertinimą ir trukmę.
## Programos funkcionalumas
- Peržiūra - visi filmai rodomi lentelėje
- Pridėjimas - Įvedamas naujas filmas su visais laukais
- Redagavimas - Keičiami pasirinkto filmo duomenys
- Trinimas - Pašalinamas pasirinktas filmas
- Rikiavimas - Išdėstomas filmų sąrašas pagal pavadinimą, metus, įvertinimą arba trukmę
- Filtravimas - Rodomi filmai, kurie atitinka nustatytus kriterijus (Žanras, metai nuo-iki, minimalus įvertinimas)
## Naudojami failai
- main.cpp - programos kodas
- filmai.txt - filmų duomenų failas
## Paleidimo instrukcija
- Paleidžiamas naudojant jūsų pasirinktą kompiliatorių.
- Failas "filmai.txt" turi būti tame pačiame aplanke kaip ir programa.
